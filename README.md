### Swagger UI

#### edit the yaml

- change the follow environments

```yaml
  OAUTH_CLIENT_ID: your-client-id
  OAUTH_CLIENT_SECRET: your-secret
  CONFIG_URL: http://localhost:9301/api.json
  URL: http://localhost:9301/api.json
```

#### run

```shell
docker-compose up 
```
